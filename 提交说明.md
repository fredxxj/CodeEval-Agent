# 附录提交代码说明

本目录用于论文、申报或课程材料中的代码附录提交，已从项目目录中整理出可阅读、可复现的核心源码。

## 目录结构

```text
附录提交代码/
├── README.md
├── 提交说明.md
├── pythonProject7/
│   ├── .env.example
│   ├── generate.py
│   ├── grading_agent.py
│   ├── main.py
│   ├── requirements.txt
│   └── resume_failed.py
└── 前端/
    ├── README.md
    ├── app.js
    ├── index.html
    └── styles.css
```

## 包含内容

- `pythonProject7/`：编程作业自动批改主流程代码，包括 Word 作业读取、学生信息提取、大模型评分、结果解析、Excel 导出和失败项续跑。
- `前端/`：项目演示用前端页面源码，可通过后端协同服务打开。
- `README.md`：项目运行方式和环境变量配置说明。
- `pythonProject7/.env.example`：环境变量示例，不包含真实 API Key。

## 未包含内容

以下内容属于本地数据、运行产物或编辑器配置，未放入附录提交目录：

- 学生作业文件、评分结果 Excel 文件。
- `.env` 等包含密钥的本地配置文件。
- `.idea/`、`__pycache__/`、`.DS_Store` 等编辑器和系统缓存。
- PDF、PPT、Word、ZIP、截图等非源码材料。
- `111.py`、`merge.py` 等临时分析或本地合并脚本。

## 运行方式

后端批改流程：

```bash
cd pythonProject7
python3 -m pip install -r requirements.txt
export BIGMODEL_API_KEY="你的智谱 BigModel API Key"
python3 main.py "/path/to/student/docx/folder" "评分结果.xlsx"
```

前后端协同服务：

```bash
cd pythonProject7
export BIGMODEL_API_KEY="你的智谱 BigModel API Key"
python3 server.py
```

然后访问 `http://localhost:8080`。
