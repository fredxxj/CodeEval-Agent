# 编程作业评估智能体前端

这是面向“创AI案例”申报与现场演示的前端界面。当前版本已可与 `pythonProject7/server.py` 协同运行，点击页面按钮即可触发后端批改流程。

## 展示内容

- 学生试题位置导入：展示本地 Word 作业目录和答题记录样例。
- 阅卷尺度：支持低、中、高三档评分尺度展示。
- 开始改卷：调用后端接口启动真实批改流程。
- 进度查看：实时展示后端进度、当前文件和运行日志。
- 结果查看：读取后端生成的 `评分结果.xlsx`，展示匿名化评分摘要。

## 运行方式

请在后端目录启动协同服务：

```bash
cd ../pythonProject7
export BIGMODEL_API_KEY="你的智谱 BigModel API Key"
python3 server.py
```

然后访问：

```text
http://localhost:8080
```

## 数据说明

页面读取真实结果表，但展示时默认使用“学生01、学生02”等匿名标识，避免在申报展示时暴露学生姓名和学号。
